# P-gina-web
Pagina de maquetacion web sobre las universidades
# Directorio de 12 Universidades de México

Este es un proyecto escolar de Maquetación Web Interactiva desarrollado con tecnologías nativas estandarizadas. Permite explorar e investigar de forma dinámica las ventajas y áreas de oportunidad de distintas instituciones de educación superior en el país.

## Características Incorporadas
* **Diseño Responsivo:** Adaptado completamente a dispositivos móviles y de escritorio mediante CSS Grid y Flexbox.
* **Buscador en Tiempo Real:** Filtrado interactivo instantáneo mediante manipulaciones del DOM con JavaScript.
* **Interfaz de Usuario Fluida:** Modales interactivos, menús colapsables y botones con animaciones dinámicas (*Ripple Effect*).
* **Fácil Navegación:** Identificación rápida por categorías (*Pública/Privada*) e inclusión de iconos estilizados.

## Tecnologías Utilizadas
* HTML5 Semántico
* CSS3 Avanzado (Animaciones, Variables y Grid Layout)
* JavaScript Nativo (ES6+)
